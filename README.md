# AI Travel & Expense Copilot

A production-style AI backend project for automating travel and expense workflows using FastAPI, AI agents, policy validation, receipt parsing, and analytics.

## Why this project?

This project demonstrates backend engineering, AI workflow orchestration, RAG-based policy checks, and analytics for enterprise travel and expense systems.

## Features

- AI-powered expense claim review
- Travel and expense policy validation
- Receipt parsing and structured extraction
- RAG-style policy retrieval
- FastAPI REST APIs
- ClickHouse analytics schema
- Docker-based local setup
- Unit tests
- GitHub Actions CI pipeline

## Tech Stack

- Python
- FastAPI
- Pydantic
- LangGraph-style workflow design
- RAG and semantic search concepts
- ClickHouse
- Docker
- GitHub Actions
